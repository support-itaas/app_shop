Account Cash Flow Statement Report in PDF & Excel Format

This module allow you to print Cash Flow Statement. For your existing chart of accounts you have to setup configuration by going to every accounts which should be included on Cash Flow Statement and if you add new account in your chart of accounts you have to set up same things. This report is totally based on accounts you configured for Cash Flow Statement. All accounts configured for Cash Flow will be automatically add up on respected Financial Report selected on that account.
